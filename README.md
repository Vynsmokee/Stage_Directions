# Stage Directions

A modern landing page built with **Angular** and **Tailwind CSS**.

## Tech Stack

- [Angular](https://angular.dev/) — Component-based frontend framework
- [Tailwind CSS](https://tailwindcss.com/) — Utility-first CSS framework

## Getting Started

### Prerequisites

- Node.js >= 18
- npm >= 9
- Angular CLI: `npm install -g @angular/cli`

### Installation

```bash
npm install
```

### Development Server

```bash
ng serve
```

Navigate to `http://localhost:4200/`.

### Build

```bash
ng build
```

Output will be in the `dist/` folder.
